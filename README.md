# Getting Started

**Clone the repo**

```sh
git clone https://github.com/CristinaFores/cart
```

**Install NPM packages**

```sh
npm install
```

**Run the project**

```sh
npm start
```

- Then, open [http://localhost:3000](http://localhost:3000)
